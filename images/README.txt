# Dossier des images

Place ici toutes les photos de bactéries (Gram, colonies, tests biochimiques)
pour qu'elles apparaissent dans les fiches.

## Recommandations techniques

- **Format** : JPEG (.jpg) de préférence, PNG (.png) accepté
- **Taille** : maximum 1000 pixels sur le plus grand côté
- **Poids** : moins de 200 Ko par image
- **Compression** : utiliser TinyPNG (https://tinypng.com) avant de déposer
  → divise le poids par 3 à 5 sans perte visible de qualité

## Nommage des fichiers

Utilise des noms en minuscules, sans espaces ni accents :

  ✓ staphylococcus_aureus_gram.jpg
  ✓ staphylococcus_aureus_colonies_sang.jpg
  ✓ ecoli_macconkey.jpg

  ✗ Staph Aureus Gram.JPG          (majuscules et espaces)
  ✗ colonies_β_hémolytiques.jpg    (caractères spéciaux)

## Comment ajouter une image à une bactérie

1. Dépose le fichier dans ce dossier
2. Ouvre l'application → icône engrenage (admin) → mot de passe
3. Onglet "Bactéries" → choisis la bactérie → "Modifier"
4. Dans le champ "Images de la bactérie", tape le chemin :
   
   images/staphylococcus_aureus_gram.jpg | Coloration Gram, amas de coques
   
   Format : `chemin | légende` (la légende après la barre verticale est optionnelle).
5. Une bactérie peut avoir plusieurs images — une par ligne dans le champ.
6. Clique sur "Enregistrer"
7. Ouvre la fiche de la bactérie → les images apparaissent dans une galerie
8. Clic sur une miniature → l'image s'ouvre en grand (lightbox)

## Images non trouvées

Si le chemin est incorrect ou l'image manquante, la galerie affichera une
vignette grise avec "⚠ image introuvable" et le chemin problématique. 
Utile pour corriger les fautes de frappe.
